# JWT token

```json
{
  "tenantId": "uuid", // Which university/institution
  "userId": "uuid", // User's ID
  "role": "STUDENT", // STUDENT | FACULTY (B2S only) | UNIVERSITY_ADMIN | COMPANY
  "tenantType": "CREATIVE_NETWORK" // Which product interface (determines UI)
}
```

# Definitions

roles: `STUDENT | FACULTY | UNIVERSITY_ADMIN | COMPANY | ADMIN | BRAND`
tenantType: `CREATIVE_NETWORK | ACADEMIC_INSTITUTION | FAN_MARKETPLACE`
